Reserve Table

Reserve Table

Reserve Table is a sleek and user-friendly table reservation system designed to simplify the dining experience. With this application, users can effortlessly browse and select restaurants, view their meal offerings, and book seats for their desired date and time. Built with the MERN stack (MongoDB, Express.js, React, Node.js) and enhanced with Redux for state management and Axios for seamless API calls, Reserve Table ensures a smooth and efficient reservation process from start to finish.

Shoehub E-commerce Website

Shoehub is a dynamic e-commerce website crafted to provide a seamless shopping experience for footwear enthusiasts. Developed using EJS, Node.js, MongoDB, and Express, this platform allows users to effortlessly browse and purchase products, add items to their wishlist, and manage their shopping cart. With advanced filtering options and a referral system, Shoehub makes finding the perfect pair of shoes easy and enjoyable.
